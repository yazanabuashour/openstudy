# OpenStudy Eval Results

This directory is for reduced OpenStudy eval reports generated by:

```bash
mise exec -- go run ./scripts/agent-eval/os7nh run
```

Committed reports must be sanitized summaries only. They may reference raw logs
or run roots only with placeholders such as
`<run-root>/production/rough-card-create/events.jsonl`.

Do not commit raw logs, SQLite databases, generated run roots, secrets, or
machine-local paths.
